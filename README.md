guice-netty-bootstrap
=====================

Guice 3 and Netty 4 Sample Project

Dependencies included
---------------------
- Guice 3.0
- Guava (Latest release)
- Netty 4.0.23.Final
- JUnit 4.10

Requirements
------------
- Java 7
- Maven 3

Running with Maven
--------
- Make the war file <code> mvn exec:java </code>
